# _✨ 人物画像插件 ✨_  

## ⌨️ 指令表

|     指令      |                    说明                    |
|:-------------:|:-----------------------------------------------:|
| 画像@群友       | 分析这位群友的性格画像，如果不指定，则分析消息发送者 |
| 画像提示词 <命令/留空> | 查看某套提示词的内容， 不指定命令则默认查看所有提示词                    |
